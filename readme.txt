== Description ==

This plugin serves a custom robots.txt file from a configurable path for sites using The SEO Framework. It allows admins to customize the robots.txt path and view the current contents.

== Installation ==

1. Upload `seo-framework-custom-robots-txt` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Ensure The SEO Framework (slug: `autodescription`) is installed and activated.
4. Configure the output path and view the robots.txt contents under **Settings > Robots.txt Settings**.

== Frequently Asked Questions ==

= What does this plugin require? =
This plugin requires WordPress 6.5 or higher and The SEO Framework plugin (slug: `autodescription`).

= Where can I set the robots.txt path? =
Go to **Settings > Robots.txt Settings** in the WordPress admin dashboard.
